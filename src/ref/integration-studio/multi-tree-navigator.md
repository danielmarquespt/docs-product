# Multi-tree Navigator

The **Multi-tree Navigator** is the area of your [workspace](https://github.com/danielmarquespt/docs-product/tree/e7ea3f444d5129dab245c69ab72ae091554bc4fb/src/ref/integration-studio/workspace.md%3E) where you can see the elements and resources associated with your extension. The Multi-tree navigator has two tabs: Extension and Resources.

Extension tab : Tree with the [actions](https://github.com/danielmarquespt/docs-product/tree/e7ea3f444d5129dab245c69ab72ae091554bc4fb/src/extensibility-and-integration/integration-studio/managing-extensions/action-define.md%3E), [entities](https://github.com/danielmarquespt/docs-product/tree/e7ea3f444d5129dab245c69ab72ae091554bc4fb/src/extensibility-and-integration/integration-studio/managing-extensions/entity-define.md%3E), and [structures](https://github.com/danielmarquespt/docs-product/tree/e7ea3f444d5129dab245c69ab72ae091554bc4fb/src/extensibility-and-integration/integration-studio/managing-extensions/structure-define.md%3E) defined in the current extension, organized in separate folders. With a right-click on these folders, you have the options of creating and deleting an element.

Resources tab : Tree with the resources associated with the current extension. For more information, see [Resources](https://github.com/danielmarquespt/docs-product/tree/e7ea3f444d5129dab245c69ab72ae091554bc4fb/src/ref/integration-studio/resources-tree.md%3E) tree.
